# JetMyMoto Platform

## Overview
JetMyMoto is a motorcycle logistics platform (NOT a transport company).

Core architecture:
- React (Vite) SPA
- Firebase backend (Cloud Functions + Firestore)
- Shared routing tree across JetMyMoto and RiderAtlas
